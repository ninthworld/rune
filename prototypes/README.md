# Prototypes

Standalone, dependency-free HTML prototypes from the design phase. They are the
living reference for UI behavior and the card render spec.

- ui-battlefield-v3.html — Pixi card factory, battlefield bands, collapsed
  stacks, subject-owned actions, zone rail, graveyard browser, peek/pinned
  inspect. Open directly in a browser.

**Reference only.** Production code reimplements these patterns inside
clients/web with proper structure and tests — never import from this directory.
